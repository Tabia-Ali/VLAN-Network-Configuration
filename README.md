# VLAN Network Configuration

## 📖 Project Overview

This project demonstrates VLAN segmentation using Cisco Packet Tracer. Two VLANs (VLAN 10 and VLAN 20) are configured on separate switches and connected through a central switch and router.

---

## 🧠 Network Design

* VLAN 10 → PC0, PC1
* VLAN 20 → PC2, PC3
* Multiple switches connected
* Router used for inter-VLAN communication

---

## 🎯 Key Features

* Logical network segmentation
* Improved security using VLANs
* Scalable network design
* Inter-VLAN routing concept

---

## 🛠 Tools Used

* Cisco Packet Tracer
